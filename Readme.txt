It's a simply program in which you'll be able to get the Hash of a file gived as input and then iterate the hash, using the output hash as a input string plus an aleatory string.
The program create a Sqlite file to serve as a route to replicate the process. Therefore, it allow validate the original file thanks to the sqlite file which behaves as a key.
The propose of this program is give a strong security of the hash to determinated uses, as publish in public the hash code of a commercial secret.
